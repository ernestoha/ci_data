# ci_data
CodeIgniter form to check and upload data using Ajax.

### Examples
#### From Browser
`http://localhost/ci_data/`

### Structure of config.php
Update **config.php** with the CI application URL and PORT:
```
$config['base_url'] = 'http://localhost/ci_data/';
```
